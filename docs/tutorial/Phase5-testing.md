# Phase 5: 테스트 및 검사 (Testing)

> 예상 시간: 1시간
> 선행 조건: Phase 1, 2, 3, 4 완료

## 작업 목록

### 5.1 HelpModal 렌더링 테스트

**테스트 항목**:
- [ ] 모달이 정상적으로開く
- [ ] 4개 탭 (개요/부품/조작법/맵) 전환 정상
- [ ] X 버튼으로 닫기 가능
- [ ] Esc 키로 닫기 가능
- [ ] 오버레이 클릭으로 닫기 가능

**테스트 방법**:
```bash
bun run dev
# 브라우저에서 http://localhost:5173 접속
# 메뉴 → 도움말 클릭
# 각 탭 클릭하여 전환 확인
```

### 5.2 온보딩 힌트 시점 테스트

**시나리오 1: 첫 방문 (튜토리얼 미완료)**
- [ ] 게임 시작 시 "설계 화면에서 D를 눌러..." 힌트 표시
- [ ] 부품 선택 시 "빨간 마나와 회로를 배치하세요" 힌트 표시
- [ ] 저장 시 "전투 시작 버튼을 눌러..." 힌트 표시
- [ ] 전투 시작 시 "키 1~5로 술식을 발사하세요" 힌트 표시

**시나리오 2: 재방문 (튜토리얼 완료)**
- [ ] 게임 시작 시 힌트 미표시 (tutorialSeen = true)
- [ ] 모든 시점에서 힌트 미표시

**테스트 데이터 초기화**:
```javascript
// 브라우저 Console에서
localStorage.removeItem('jesulkr_tutorial_seen_v2');
localStorage.removeItem('magic_design_game_slots_v2');
// 페이지 새로고침
```

### 5.3 다국어 (ko/en) 전환 테스트

**테스트 항목**:
- [ ] 한국어에서 도움말 메뉴 확인
- [ ] English에서 도움말 메뉴 확인
- [ ] 힌트 메시지 한국어/English 적절히 전환
- [ ] 부품 설명 한국어/English 적절히 전환

**테스트 방법**:
```bash
# 메뉴 → Language → English 클릭
# 도움말 → 부품 탭에서 모든 부품 설명 확인
# 다시 한국어切换
```
